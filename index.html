A* PathFinding

Aim

The aim of this Virtual Lab is to implement the A* pathfinding algorithm that dynamically finds the shortest path between two points in a city, combining heuristic and cost-based approaches to efficiently navigate through a weighted graph, similar to real-world navigation systems.

Theory

The A* pathfinding algorithm is a widely used search algorithm that combines the strengths of Dijkstra's algorithm and Greedy Best-First Search. It uses a heuristic function to estimate the cost from a given node to the goal, alongside the actual cost from the start node to the current node, to prioritize paths that are most likely to lead to the shortest solution. This blend of cost-based and heuristic-driven exploration makes A* highly efficient in navigating complex graphs.

Pathfinding algorithms, such as A*, are fundamental concepts in computer science and have diverse applications in solving real-world problems.
Its some applications include:

  • Navigation Systems: A* is commonly used in GPS navigation systems to calculate the shortest route between locations.
  • Game Development: The algorithm helps game characters or objects find the optimal path through game levels or environments.
  • Robotics: Robots use A* to navigate through physical spaces while avoiding obstacles.
  • Network Routing: A* is employed in network protocols to find the shortest or most efficient data transmission paths.
  • Urban Planning: A* assists in planning efficient routes for transportation and logistics in urban environments.
Procedure

The procedure for implementing the A* pathfinding algorithm can be summarized as follows:

  1. Input the graph representation (nodes, edges, and weights) and the start and goal nodes.
  2. Initialize the following data structures:
    1. A priority queue to store nodes, prioritized by their total estimated cost (f = g + h).
    2. A dictionary to track the actual cost (g) of reaching each node from the start node.
    3. A heuristic function (h) that estimates the cost from any node to the goal.
    4. A dictionary to keep track of parent nodes for reconstructing the path.
  3. Set the g-value of the start node to 0 and calculate its f-value using the heuristic function. Add the start node to the priority queue.
  4. While the priority queue is not empty:
    1. Dequeue the node with the lowest f-value from the priority queue.
    2. If the dequeued node is the goal node, terminate the search and reconstruct the path using the parent nodes.
    3. For each neighboring node of the current node:
      1. Calculate the tentative g-value for the neighbor as the sum of the g-value of the current node and the edge cost.
      2. If the tentative g-value is lower than the recorded g-value for the neighbor or the neighbor is not in the priority queue:
        1. Update the g-value and calculate the f-value (f = g + h) for the neighbor.
        2. Set the current node as the parent of the neighbor.
        3. Add or update the neighbor in the priority queue with the new f-value.
  5. If the priority queue becomes empty and the goal node is not reached, conclude that no path exists.
  6. Output the reconstructed shortest path and its total cost if a path is found.

// C++
#include < iostream >
using namespace std;
 
int linearSearch(const int* arr, int size, int target) {
    for (int i = 0; i < size; ++i) {
        if (arr[i] == target)
            return i;  // Found the target
    }
    return -1;  // Target not found
}
int binarySearch(const int* arr, int size, int target) {
    int left = 0;
    int right = size - 1;
 
    while (left <= right) {
        int mid = left + (right - left) / 2;
 
        if (arr[mid] == target)
            return mid;  // Found the target
 
        if (arr[mid] < target)
            left = mid + 1;
        else
            right = mid - 1;
    }
 
    return -1;  // Target not found
}
struct Search{
    /*
    Defines a structure named 'Search' that is used to represent search results or items.
    The 'Search' structure has two members:
 
    'index' (integer): Represents the index or position of a search result or item.
    'sorted' (integer): Indicates whether the search result or item is sorted.
    This structure can be utilized to organize and store information
        related to search operations, such as maintaining the index and sorted
        status of search results in a list.
    */
        int index;
        int sorted;
    };
Search adaptiveSearch(const int* arr, int size, int target){
 
    Search searchResult;
    searchResult.sorted =0;
    searchResult.index =-1;
    // Check if the list is sorted
    bool isSorted = true;
    for (int i = 1; i < size; ++i) {
        if (arr[i] < arr[i - 1]) {
            isSorted = false;
            break;
        }
    }
    
    if (isSorted){
        searchResult.index =  binarySearch(arr, size, target);
        searchResult.sorted = 1;
        return  searchResult;}
    else{
        searchResult.index =  linearSearch(arr, size, target);
        searchResult.sorted = 0;
        return  searchResult;}
}

# Python
def linear_search(arr, target):
    """
    Performs linear search on the given list to find the target element.
 
    Args:
        arr (list): The list to be searched.
        target: The element to be found in the list.
 
    Returns:
        int: The index of the target element if found, -1 otherwise.
    """
    for i in range(len(arr)):
        if arr[i] == target:
            return i  # Target element found, return the index
    return -1  # Target element not found
 
 
def binary_search(arr, target):
    """
    Performs a binary search on the given sorted list to find the target element.
    Args:
        arr (list): The sorted list to be searched.
        target: The element to be found in the list.
    Returns:
        int: The index of the target element if found, -1 otherwise.
    """
    left, right = 0, len(arr) - 1
    while left <= right:
        mid = (left + right) // 2
        if arr[mid] == target:
            return mid
        elif arr[mid] < target:
            left = mid + 1
        else:
            right = mid - 1
    return -1
 
def main():
    # Take input from the user
    arr = input("Enter the list (space-separated integers): ").split()
    arr = [int(x) for x in arr]
    target = int(input("Enter the target element: "))

    # Determine if the list is sorted
    is_sorted = sorted(arr) == arr

    # Perform the appropriate search
    if is_sorted:
        print("Sorted list given.\nPerforming Binary Search.")
        index = binary_search(arr, target)
        if index == -1:
            print(f"The target {target} is not found in the sorted list.")
        else:
            print(f"The target {target} is found at index {index} in the sorted list.")
    else:
        print("Unsorted list found.\nPerforming Linear Search.")
        index = linear_search(arr, target)
        if index == -1:
            print(f"The target {target} is not found in the unsorted list.")
        else:
            print(f"The target {target} is found at index {index} in the unsorted list.")
